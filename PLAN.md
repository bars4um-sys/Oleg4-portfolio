# План проекта "Моё Портфолио" (oleg4.ru)

## 🎯 Цель
Создать систему сайтов-портфолио на Jekyll, где каждый навык представлен отдельным лендингом, объединённых общей навигацией и дизайн-системой.

---

## 📦 Технический стек
- **Генератор:** Jekyll 4.4.1
- **Ruby:** 2.6.10
- **Хостинг:** Timeweb (VPS или обычный хостинг)
- **Домен:** oleg4.ru

---

## 📋 Этапы реализации

### Этап 1: Инициализация проекта
- [ ] Создать Jekyll-проект в папке `MyPortfolio`
- [ ] Настроить `_config.yml` (домен, URL, базовые параметры)
- [ ] Подключить необходимые гемы (jekyll-seo-tag, jekyll-sitemap и др.)
- [ ] Настроить Gemfile

### Этап 2: Дизайн-система
- [ ] Разработать цветовую палитру (основной цвет, акценты, фон, текст)
- [ ] Выбрать шрифты (Google Fonts: Montserrat + Roboto Slab и др.)
- [ ] Создать логотип (текстовый/графический SVG)
- [ ] Создать единые CSS-компоненты (шапка, подвал, карточки, кнопки)
- [ ] Реализовать уникальные CSS-темы для каждого навыка

### Этап 3: Структура контента
```
/
├── index.html              # Главная — агрегатор всех навыков
├── about.md                # Обо мне
├── contacts.md             # Контакты
├── blog/                   # Блог (Jekyll built-in)
├── skills/
│   ├── landing-pages/      # Создание лендингов
│   ├── photo-restoration/  # Восстановление фото
│   ├── photobooks/         # Фотокниги
│   ├── ai-avatars/         # AI-аватары
│   ├── neuro-photoshoot/   # Нейрофотосессии
│   ├── custom-songs/       # Адресные песни
│   └── software/           # Программные продукты
└── assets/
    ├── css/
    ├── js/
    └── images/
```

### Этап 4: Создание страниц навыков (лендингов)
Каждый лендинг содержит:
- **Hero-секция** с уникальным дизайном
- **Описание услуги** (что я предлагаю)
- **Портфолио/кейсы** (примеры работ)
- **Отзывы клиентов**
- **Цены/тарифы** (опционально)
- **CTA-блок** (призыв к действию — заказать)

### Этап 5: Блог
- [ ] Несколько категорий/рубрик (по навыкам)
- [ ] Возможность создания нескольких блогов (через коллекции)

### Этап 6: Дополнительные страницы
- [ ] **Обо мне** — биография, опыт, философия
- [ ] **Контакты** — форма обратной связи, ссылки на мессенджеры/соцсети
- [ ] **Отзывы** — отдельная страница-сборник

### Этап 7: Технические штуки
- [ ] SEO-оптимизация (мета-теги, Open Graph, sitemap.xml)
- [ ] Адаптивность (мобильная версия)
- [ ] Производительность (минификация CSS/JS)
- [ ] Яндекс.Метрика / Google Analytics

### Этап 8: Деплой на Timeweb
- [ ] Настройка деплоя на VPS (через Git + webhook или вручную)
- [ ] Подключение домена oleg4.ru
- [ ] SSL-сертификат (Let's Encrypt)

---

## 🔥 Приоритет на первый запуск
1. Сделать **главную страницу** + **2-3 лендинга** (по самым сильным навыкам)
2. Страницы **"Обо мне"** и **"Контакты"**
3. **Блог** с первой публикацией
4. Залить на сервер

## ❓ Открытые вопросы
1. **Логотип** — текстовый (шрифт) или графика (SVG)?
2. **Цветовая схема** — предпочтения? (тёмная/светлая тема, холодные/тёплые тона)
3. **Какой навык самый приоритетный?** С какого лендинга начнём?
4. **Готовый сайт-портфолио** — ссылка или структура для интеграции контента
5. **VPS или обычный хостинг Timeweb?** Для Jekyll подойдёт обычный хостинг, VPS даёт больше гибкости
